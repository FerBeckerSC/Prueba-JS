JavaScript

Este lenguaje de programacion trabaja junto a HTML y CSS

Hay tres formas de vincular HTML y JS

  • En Linea: Escribo el codigo JS en la etiqueta de apertura (no se recomienda)
  • Interna:Escribo el codigo entre las etiquetas de script
  • Externa:Escribo el codigo JS en un archivo diferente y lo vinculo a traves de la etiqueta de script
